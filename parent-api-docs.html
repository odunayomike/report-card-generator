üìñ Overview

The SchoolHub Parent API provides secure access for parents and guardians to view their children's academic performance, attendance, and behavioral data through a mobile application.

Base URL: https://your-domain.com/backend/routes/parent/

‚ú® Key Features

  • Email-only authentication (no password required)
  • Session-based authentication with cookies
  • Comprehensive academic analytics
  • Performance history and trends
  • Real-time attendance tracking
  • Behavioral and skills assessment

üîê Authentication

The API uses email-only authentication with session-based cookies. No password is required.

Always include credentials: 'include' in all API requests to maintain session state.

üíª Code Examples

JavaScript / React Native

const API_BASE_URL = 'https://your-domain.com/backend/routes/parent';

// Login
async function login(email) {
  const response = await fetch(`${API_BASE_URL}/login.php`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    credentials: 'include',
    body: JSON.stringify({ email })
  });
  return await response.json();
}

// Get Children
async function getChildren() {
  const response = await fetch(`${API_BASE_URL}/get-children.php`, {
    credentials: 'include'
  });
  return await response.json();
}

// Get Child Analytics
async function getChildAnalytics(studentId) {
  const response = await fetch(
    `${API_BASE_URL}/get-child-analytics.php?student_id=${studentId}`,
    { credentials: 'include' }
  );
  return await response.json();
}

Flutter / Dart

import 'package:http/http.dart' as http;
import 'dart:convert';

class ParentApi {
  static const String baseUrl =
    'https://your-domain.com/backend/routes/parent';

  static Future<Map<String, dynamic>> login(String email) async {
    final response = await http.post(
      Uri.parse('$baseUrl/login.php'),
      headers: {'Content-Type': 'application/json'},
      body: jsonEncode({'email': email}),
    );
    return jsonDecode(response.body);
  }

  static Future<Map<String, dynamic>> getChildren() async {
    final response = await http.get(
      Uri.parse('$baseUrl/get-children.php'),
    );
    return jsonDecode(response.body);
  }
}

  • Use the same session across all requests
  • Test authentication first before other endpoints
